## Getting Started

This is an example of how set up this project locally.
To get a local copy up and running follow these simple steps.

### Prerequisites

- Node.js
- npm ou yarn

### Installation

1. Clone the repo
   ```sh
   git clone
   ```
2. Install NPM packages
   ```sh
   yarn
   ```
   or
   ```sh
   npm install
   ```
3. Run project
   ```sh
   yarn dev
   ```
   or
   ```sh
   npm run dev
   ```
